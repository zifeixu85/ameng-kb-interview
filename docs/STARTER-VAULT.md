# Starter Vault 使用说明

这份压缩包只适用于新建或空的 Obsidian Vault。

1. 解压后打开 `starter-vault/`。
2. 在 Obsidian 里选择“打开文件夹作为仓库”。
3. 在 Codex 中打开完全相同的 `starter-vault/`。
4. 安装 `ameng-kb-interview` Skill。
5. 输入：`使用 $ameng-kb-interview 帮我完成首次采访建库。`

请先看目录预案，确认后再写入。已有 Vault 不要手动合并这份模板，而应使用 Skill 的只读映射流程。

可选择：快速版（先跑通）、标准版（建立核心首版）或分多次完成的深度版。Starter Vault 不会预建完整自我说明书空目录；采访确认某个模块有真实内容后再创建。

`private / internal / public` 是 AI 协作策略标签，不是加密或访问控制。不要在 Vault 中保存密码、Token 或其他秘密。
