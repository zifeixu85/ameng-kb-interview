# 自我说明书

这里记录帮助 AI 理解你的真实上下文。它是“真实底层”，不是用于公众号、小红书、简历或商业介绍的对外人设。

完整方法论包含：

```text
01-身份与经历
02-价值观与驱动力
03-性格底色与心理测试
04-偏好审美与表达习惯
05-身体健康与能量状态
06-家庭亲密关系与责任
07-社会生活与重要关系
08-状态变化与阶段复盘
09-日记与自我观察
10-AI协作偏好
99-隐私边界与公开授权
```

这些目录根据采访和已有证据渐进创建，不要求第一次全部出现。心理、健康、家庭、关系和日记等敏感模块必须由你明确选择是否启用；也可以只记录对工作与决策有影响的边界，不记录私人细节。

本目录默认 `private`。建库、自我梳理和个人协作任务可以定向读取完成任务所需的正文；这不等于公开授权。公开写作或对外介绍应使用经过拆分和确认的对外版本，不直接读取这里的完整细节。
